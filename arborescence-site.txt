ARBORESCENCE DU SITE WEB
========================

Ce fichier explique l'organisation du site web des cabinets d'osteopathie du Pian-Medoc.


HIERARCHIE VISIBLE DU SITE
--------------------------

Le site est organise autour d'une page d'accueil et de plusieurs pages accessibles depuis le menu principal.

Menu principal du site :

1. Accueil
   Fichier : index.html
   Role : page d'entree du site.
   Elle presente les deux cabinets, les praticiens, les liens de rendez-vous et les principaux boutons de navigation.

2. L'osteopathie
   Fichier : osteopathie.html
   Role : page d'explication.
   Elle presente ce qu'est l'osteopathie, son approche et les benefices possibles.

3. Cabinet Celine
   Fichier : celine.html
   Role : page du cabinet de Celine Andurand.
   Elle presente Celine Andurand, Kim Sabourin, Laura Loison, le Pole Sante Victor Hugo et les informations de rendez-vous.

4. Cabinet Bryan
   Fichier : bryan.html
   Role : page du cabinet de Bryan Salmon.
   Elle presente Bryan Salmon, Antoine Cazot, la Maison Medicale Portes du Medoc et les informations de rendez-vous.

5. Consultations
   Fichier : consultations.html
   Role : page des consultations.
   Elle explique les motifs de consultation, les publics concernes et les informations pratiques.

6. Contacts
   Fichier : contacts.html
   Role : page de contact.
   Elle regroupe les coordonnees, les adresses, les telephones et les liens de rendez-vous.

7. Prendre RDV
   Lien externe : https://www.doctolib.fr
   Role : bouton de prise de rendez-vous visible dans le menu.

Parcours utilisateur principal :

1. L'utilisateur arrive sur index.html.
2. Il choisit un cabinet ou une rubrique depuis le menu.
3. Il peut consulter les informations de Celine, de Bryan, des consultations ou de contact.
4. Il peut prendre rendez-vous avec les liens Doctolib ou appeler directement les cabinets.


RACINE DU SITE
--------------

ost-opathes-lepianmedoc.fr/

C'est le dossier principal du site. Il contient toutes les pages HTML, les fichiers de style,
les scripts et les images utilisees par le site.


FICHIERS PRINCIPAUX
-------------------

index.html
Page d'accueil du site.
Elle presente les cabinets, les praticiens, les liens de rendez-vous et les boutons principaux
vers l'osteopathie, les consultations et les pages des cabinets.

osteopathie.html
Page d'explication de l'osteopathie.
Elle sert a presenter la discipline, son approche et ses benefices.

consultations.html
Page dediee aux consultations.
Elle explique les motifs de consultation, les publics concernes et les informations pratiques.

celine.html
Page du cabinet de Celine Andurand.
Elle presente Celine, ses collaboratrices, le cabinet et les informations de rendez-vous.

bryan.html
Page du cabinet de Bryan Salmon.
Elle presente Bryan, son collaborateur, le cabinet et les informations de rendez-vous.

contacts.html
Page de contact.
Elle regroupe les coordonnees, les liens de rendez-vous et les informations utiles pour joindre les cabinets.

style.css
Feuille de style principale du site.
Elle gere l'apparence de toutes les pages : couleurs, typographies, espacements, boutons,
navigation, sections, mise en page responsive, etc.

script.js
Fichier JavaScript du site.
Il gere les interactions simples, comme le menu mobile, les effets au scroll ou les animations.

projet.md
Document de travail du projet.
Il contient des notes, textes ou informations de reference pour construire le site.

NOTES IMPORTANTES
-----------------

- Les pages HTML sont les pages visibles du site.
- Le fichier style.css controle l'apparence globale.
- Le fichier script.js controle les interactions.
- Les dossiers SERVICES et LOGO contiennent les images utilisees par le site.
- Pour modifier un texte visible sur le site, il faut generalement ouvrir la page HTML concernee.
- Pour modifier les couleurs, les tailles, les espacements ou la mise en page, il faut modifier style.css.
- Pour remplacer une image, il faut changer le fichier dans SERVICES ou LOGO, ou modifier le chemin dans la page HTML.
